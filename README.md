# spark-blur-image
This is related to cloud computing course to blurring image using spark in python (pyspark).

## Pre-requirement:
- openjdk (17.0.5)
- python (3.10)
- pip

## Setup the project
```pip install -r requirements.txt```

it need a jpg image file with "image.jpg" name
